### 퓨처스리그 앱 사용성 데이터를 통한 대출신청 예측분석

- 가명화된 데이터를 기반으로 고객의 대출상품 신청여부 예측
(2022년 3~5월 데이터제공 / 2022년 6월 예측)
- 예측모델을 활용하여 탐색적 데이터 분석 수행

- 대출신청, 미신청 고객을 분류하여 고객의 특성 분석결과 도출
